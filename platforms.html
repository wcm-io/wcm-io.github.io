wcm.io

Libraries and extensions for AEM applications

Supported Platforms

The wcm.io modules support the following platforms:

  • AEM 6.5.24+, AEM 6.6.2+, AEM Cloud Service
  • Apache Sling, Apache Felix an others: Versions as included in the corresponding AEM version
  • Runtime Environment: Java 11, Java 17, Java 21, Java 25
  • Build Environment: Java 21, Java 25
