Release 1.9.0 – not released

Type Changes By
Add AEM implementation ConfigurationInjectResourceDetectionStrategy to support correct injection of configurations in HTL and Sling Models in editable template structure components. Fixes WCON-75. sseifert

Release 1.8.8 – 2021-11-10

Type Changes By
Do not generate warnings for not allowed postfixes. Fixes WCON-86. cnagel

Release 1.8.6 – 2021-08-16

Type Changes By
Eliminate usage of Commons Collections 3. sseifert

Release 1.8.2 – 2021-05-25

Type Changes By
Add "Reference Provider Core Component Disabler" that automatically disables the reference provider implementation of the Adobe Core Components. The latter tries to achieve the same as the reference provider implementation of this module, but fails to reliably detect the last modification date of context-aware configurations stored in AEM pages. sseifert

Release 1.8.0 – 2021-01-17

Type Changes By
Switch to AEM 6.4 as minimum version. sseifert
Switch to individual OSGi artifacts with compile dependency instead of osgi.core and osgi.cmpn with provided dependency. sseifert

Release 1.7.0 – 2020-04-23

Type Changes By
Switch to AEM 6.3 as minimum version. sseifert
Use PageManagerFactory to get PageManager instances. sseifert

Release 1.6.0 – 2019-05-09

Type Changes By
Switch to AEM 6.2 as minimum version. sseifert
AEM Page persistence strategy: Ensure AEM page is created also for nested list configuration without properties on the first level. Fixes WCON-63. sseifert

Release 1.5.0 – 2019-02-22

Type Changes By
AbsoluteParentContextPathStrategy: Add 'unlimited' option to allow unlimited configuration of hierarchy levels. Fixes WCON-59. sseifert
Display contextPathRegex in webconsole hint of context path strategies. sseifert
Context path strategies: Support new version history path at /tmp/versionhistory used since AEM 6.3.2.2, 6.4.3 and 6.5. Fixes WCON-49. sseifert

Release 1.4.0 – 2018-08-30

Type Changes By
Add "resourceType" configuration property for "AEM page" persistence strategy. Fixes WCON-56. sseifert

Release 1.3.0 – 2018-05-04

Type Changes By
Add new "Template path blacklist" configuration property to AbsoluteParentContextPathStrategy. Fixes WCON-51. kwindszus
Use nt:unstructured as default primary node type for configuration collections inside cq:Page to keep item order. sseifert

Release 1.2.4 – 2018-03-14

Type Changes By
Make context path strategies compatible with side-by-side comparison in AEM 6.3. Fixes WCON-49. sseifert

Release 1.2.2 – 2017-11-13

Type Changes By
PagePersistenceStrategy: Use Page for configuration collection root resource. Fixes WCON-44. sseifert
ReferenceProvider: Return only cq:Page nodes, handle collection items correctly. Fixes WCON-45. sseifert
ReferenceProvider: Derive reference display name from config page path. Fixes WCON-46. sseifert

Release 1.2.0 – 2017-10-19

Type Changes By
PagePersistenceStrategy: Fix data loss issue when using multiple nested list configurations. Fixes WCON-39. sseifert
PagePersistenceStrategy: Use PageManager API to delete configuration pages. Fixes WCON-43. sseifert
Add Referencer Provider for Context-Aware Configuration pages. Fixes WCON-42. drudolph

Release 1.1.0 – 2017-09-12

Type Changes By
RootTemplateContextPathStrategy: Add flag templateMatchAllLevels to match all parent pages against the given template paths. Fixes WCON-37. sseifert

Release 1.0.6 – 2017-08-08

Type Changes By
PersistenceUtils: If config page has cq:lastModified this will be updated with the same value that is set to jcr:lastModified. Fixes WCON-36. ptolstych

Release 1.0.4 – 2017-05-29

Type Changes By
ToolsConfigPagePersistenceStrategy: Fix resource collection resolving when multiple bucket names are configured. sseifert
Absolute Parents Context Path Strategy: Include all resources below tools/config in blacklist pattern. sseifert
Persistence strategies: Make service ranking configurable. sseifert
Context path strategies: Pass configured service ranking to context resource. sseifert

Release 1.0.2 – 2017-05-08

Type Changes By
Apply property filter for configuration persistence provided by Apache Sling Context-Aware Configuration Management Settings. sseifert

Release 1.0.0 – 2017-04-04

Type Changes By
Initial release. sseifert
