Plugin Documentation

Goals available for this plugin:

Goal Description
cq:help Display help information on cq-maven-plugin.
Call mvn cq:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
cq:install Executes install phase and installs an OSGi bundle jar to a running Sling instance (combines goals "install" and "sling:install").

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.5.0
JDK 1.8
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>io.wcm.maven.plugins</groupId>
          <artifactId>cq-maven-plugin</artifactId>
          <version>1.1.5-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>io.wcm.maven.plugins</groupId>
        <artifactId>cq-maven-plugin</artifactId>
        <version>1.1.5-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
