i18n:transform

Full name:

io.wcm.maven.plugins:i18n-maven-plugin:1.2.1-SNAPSHOT:transform

Description:

Transform i18n resources in Java Properties, JSON or XML file format to Sling i18n Messages JSON or XML format.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: generate-resources.

Optional Parameters

Name Type Since Description
<generatedResourcesFolderPath> String - (no description)
Default value is: generated-i18n-resources.
<outputFormat> String - Output format for i18n: json, xml or properties.
Default value is: json.
<source> String - Source path containing the i18n source .properties or .xml files.
Default value is: ${basedir}/src/main/resources/i18n.
<target> String - Relative target path for the generated resources.
Default value is: SLING-INF/app-root/i18n.
