Release 2.0.4-2.25.4 – 2024-06-04

Type Changes By Update to AEM WCM Core Components 2.25.4. Fixes 22. sseifert Image (v3) and Responsive Image (v1): Set forceIgnoreFreshness property on edit dialog properties with default value (related to PR #2735). Fixes 22. sseifert Responsive Image (v1): Fix exception when trying to access asset data layer for components not based on a DAM asset (e.g. inline image). Fixes 25. sseifert

Release 2.0.2-2.23.2 – 2024-03-13

Type Changes By Download (v1/v2): Don't display file size if it is not available. Fixes 20. sseifert

Release 2.0.0-2.23.2 – 2024-01-26

Type Changes By Version 2.0.0-2.23.2 requires Handler 2.x, See Migrate from wcm.io Handler 1.x to 2.x for details. sseifert Explicitly define folder type for JSON files with "nt:unstructured" as root node to circumvent SLING-12197. sseifert Image (v3): Remove support for inline image editing, as the Adobe Image V3 component has removed support for that as well. sseifert
